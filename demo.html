excerpt.js

Quick and easy html excerpt generation to deal with dynamic content overflow.

Written by Richard Denton at eMarketeer Australia

  1. Specify your excerpt elements by using data-excerpt=[excerpt_length]
  2. Insert excerpt.js at foot - <script src="src/excerpt.js"></script></body>
  3. excerpt.js will hide original overflowing elements, and insert new, trimmed elements.
  4. This allows search engines, scrapers and indexers to still read all original content, regardless of front end design.
